APA ITU KOMUNITAS SS?

Wadah bagi para pecinta kereta api Indonesia (individu maupun organisasi). Didirikan oleh Herie Heryadi, Arjuna Pamungkas, Aldino Arya Saputra, Habibi Muhammad Arkam, dan Ahmad Suseno pada 19 Maret 2019 di Cikini, Jakarta Pusat.

VISI

Menjadi komunitas perkeretaapian terdepan yang bersatu, peduli, dan berkontribusi aktif dalam mewujudkan keselamatan, edukasi, serta budaya positif transportasi kereta api di Indonesia.

MISI

• Menumbuhkan cinta perkeretaapian
• Menjalin kemitraan dengan instansi
• Mengembangkan railfans beretika & bertanggung jawab.
