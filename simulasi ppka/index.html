CTC VDU Simulator

v2.1.0
CTC (Centralized Traffic Control) adalah simulasi menjadi Pengatur Perjalanan Kereta Api (PPKA), yang memegang kendali penuh atas keselamatan dan kelancaran operasional. Uji ketelitian dan ketangkasan Anda dalam mengatur sistem persinyalan, memindahkan wesel, serta mengeksekusi rute untuk persilangan maupun penyusulan kereta api secara real-time.
